# Ejemplo de Portafolio para Desarrollador Web

## Sobre mí

Soy desarrollador web fullstack con experiencia en automatización de reportes, dashboards interactivos y aplicaciones empresariales usando Next.js, Electron y Prisma. Me especializo en soluciones a medida para empresas que buscan eficiencia y escalabilidad.

## Tecnologías principales

- **Frontend:** Next.js, React, Material UI, CSS Modules
- **Backend:** Node.js, Express, Next.js API Routes
- **ORM/DB:** Prisma, SQL Server, PostgreSQL
- **Automatización y Reportes:** pdfmake, generación de PDFs personalizados, integración de APIs
- **Otros:** Electron, Redux Toolkit, Git, GitHub Actions, Postman

## Proyectos destacados

### 1. Sistema de Reportes Financieros PDF

- **Stack:** Next.js, Electron, Prisma, pdfmake
- **Descripción:** Automatización de reportes financieros en PDF con branding, fuentes personalizadas y filtros avanzados. Integración con base de datos y panel de administración.
- **Rol:** Fullstack developer (diseño, backend, frontend, automatización)
- **Impacto:** Reducción de tiempos de entrega en 80% y mejora en la presentación de informes.

### 2. Dashboard de Control Empresarial

- **Stack:** Next.js, Redux, Material UI
- **Descripción:** Panel interactivo para monitoreo de KPIs, gestión de usuarios y reportes en tiempo real.
- **Rol:** Frontend principal, integración con backend y APIs.

### 3. Integración de APIs y Seguridad

- **Stack:** Next.js API, Prisma, JWT, cifrado personalizado
- **Descripción:** Desarrollo de endpoints seguros, autenticación, autorización y cifrado de datos sensibles.

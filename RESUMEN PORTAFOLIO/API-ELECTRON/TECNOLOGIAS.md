# Tecnologías Utilizadas

Este proyecto de Ingreso de Artículos Masivos fue desarrollado utilizando las siguientes tecnologías y herramientas:

## Frontend

- **React**: Biblioteca principal para la construcción de interfaces de usuario interactivas y reactivas.
- **JavaScript (ES6+)**: Lenguaje de programación principal para la lógica de la aplicación.
- **HTML5**: Estructura y semántica de la aplicación web.
- **CSS3**: Estilizado y diseño responsivo de la interfaz.

## Backend y Desktop

- **Electron**: Permite empaquetar la aplicación web como una app de escritorio y acceder a funcionalidades del sistema operativo.
- **Node.js**: Entorno de ejecución para el backend y la integración con Electron.
- **MSSQL (mssql)**: Librería para la conexión y manipulación de bases de datos SQL Server desde Node.js/Electron.

## Funcionalidad destacada

- **Generación de archivos .txt**: El sistema puede generar archivos de texto (.txt) a partir de los datos procesados o cargados, facilitando la exportación y respaldo de información.
- **Carga masiva y análisis de datos**: Permite analizar y cargar grandes volúmenes de artículos a una base de datos SQL Server, asegurando integridad mediante transacciones.

## Estructura y Organización

- **Componentes reutilizables**: Uso de componentes como Dropzone, Header y Home para modularidad y mantenimiento.
- **React Router**: Para la gestión de rutas internas y navegación entre vistas.

## Herramientas de desarrollo

- **Node.js & npm**: Entorno de ejecución y gestor de paquetes para instalar dependencias y ejecutar scripts.
- **Create React App**: Herramienta para inicializar y configurar el entorno de desarrollo de React.

## Recursos adicionales

- **Imágenes y assets**: Gestión de recursos estáticos en la carpeta `src/assets`.

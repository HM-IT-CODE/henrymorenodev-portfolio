# Visor DICOM Personalizado con DC4CHE y React

Este proyecto es un visor médico DICOM basado en la plataforma OHIF, adaptado e integrado con un servidor DC4CHEE. Está diseñado para la visualización, gestión y compartición de estudios médicos en entornos clínicos.

## Características principales

- **Visualización de estudios DICOM**: Permite listar y explorar estudios almacenados en DC4CHEE.
- **Filtros y búsqueda**: Interfaz para filtrar y buscar estudios por diferentes criterios.
- **Compartir por WhatsApp**: Funcionalidad para compartir enlaces directos a estudios vía WhatsApp, facilitando la comunicación con pacientes o colegas.
- **Interfaz personalizada**: Modales y componentes de usuario mejorados usando `react-modal` y estilos modernos.
- **Integración con DC4CHEE**: Comunicación directa con el backend DC4CHEE para obtener y mostrar los estudios.

## Tecnologías utilizadas

- **React**
- **OHIF Viewer**
- **DC4CHEE**
- **react-modal**

## Personalizaciones realizadas

- Reemplazo de modales originales por `react-modal` para mayor flexibilidad y control.
- Implementación de un modal de WhatsApp para compartir enlaces de estudios.
- Mejoras visuales y de usabilidad en la lista de estudios.

## Ideal para

- Clínicas, hospitales o profesionales que requieran una solución eficiente para visualizar y compartir imágenes médicas DICOM.
